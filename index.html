English Premier League: Goal Analysis

Below is a visualization of goals scored by various EPL teams in the 2015-2016 season.

Data Source: http://openfootball.github.io/

Visual Encodings

Position: Horizontal position along the x-axis encodes two variables. The first and most prominently is opponent club. The second is time, with left being played earlier in the season. The vertical position encodes number of goals scored by the selected team.

Length/height: The length/height of the bar encodes the number of goals scored by the selected team

Hoover Text: Includes the number of goals the selected team scored in that match as well as how many were scored against them.

Reasoning: I chose to use a graph because length/height is the best way to encode discrete data (goals). From their I narrowed it down between a scatterplot and a bar chart. I felt the scatter chart did not effectively communicate an ordinal scale on the x-axis. The last consideration was how to sort the x-axis data. I chose time because "recent form" is huge in the football world. Recent form means how well the team has done the one or two games before their next. A chronological x-axis would allow the viewer to understand this concept in my visualization.


Control Features

My visualization includes two types of controls. The first type of control is the team selector. This allows the viewer to isolate the data about the football club that is most important to them. I chose to make the default Chelsea because that is my favorite club. The second control allows the user to view either home game goals or away game goals. I did this because it allows the user to ask pretty specific questions about the club they are interested in. For example, did Chelsea score more home goals later in the season? With the added hover text, a user can find an answer to the question did Chelsea run up the score on a team or was the game just generally a shoot out?
